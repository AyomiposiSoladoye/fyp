# Gemini AI Integration Setup Guide

## What is Gemini AI Integration?

Your virality prediction system now includes **Google Gemini AI** to:
- ✅ Provide AI-powered insights on predictions
- ✅ Explain why a tweet is predicted as viral/non-viral
- ✅ Generate actionable recommendations for improving virality
- ✅ Summarize model results with research-quality insights

---

## How to Setup

### Step 1: Get Your Free Gemini API Key

1. Go to: https://aistudio.google.com/app/apikey
2. Sign in with your Google account (or create one)
3. Click **"Create API Key"**
4. Copy the generated API key

### Step 2: Install Gemini Package

```bash
pip install google-generativeai
```

Or update all packages:
```bash
pip install -r requirements.txt
```

### Step 3: Add API Key to Streamlit App

1. Run the Streamlit app: `streamlit run streamlit_app.py`
2. In the left sidebar, click **"🤖 Gemini AI Setup"** to expand
3. Paste your API key in the text field
4. The system will validate it
5. You'll see ✅ "Gemini enabled! AI summaries available."

## Troubleshooting

### "Invalid API Key" Error
- Check that you copied the entire key correctly
- Make sure there are no extra spaces
- Try regenerating the key on aistudio.google.com

### "API quota exceeded"
- You've exceeded 60 requests/minute
- Wait a few minutes before making more predictions
- For high-volume use, consider upgrading to paid tier

### "ModuleNotFoundError: No module named 'google.generativeai'"
- Run: `pip install google-generativeai`

### No response from Gemini
- Check your internet connection
- Verify API key is valid
- Check Google's status page

## What Data is Sent to Google?

When you use Gemini, these are sent:
- The tweet text (for analysis)
- Feature values (numbers only, no personal data)
- Prediction results (viral/non-viral)

**NOT sent:**
- Your Gemini API key (stays local)
- Your model
- Raw dataset
- Personal information

## FAQ

**Q: Does Gemini see my personal data?**
A: No. Only tweet content and feature numbers are sent. Your model and data stay local.

**Q: Can I use this offline?**
A: No, Gemini requires internet. Predictions without Gemini still work offline.

**Q: How many predictions can I make?**
A: Unlimited, but limited to 60 per minute on free tier.

**Q: Can I cite Gemini in my thesis?**
A: Yes! It's a tool. Example: "Using Google's Gemini API for analysis..."

**Q: What if Gemini is wrong?**
A: Always verify with your own analysis. Use it as a tool, not a source of truth.
